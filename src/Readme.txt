                I- APPLICATION WEB CRUD POUR LA VENTE DES VOITURES EN LIGNE
    
Le but de ce projet est de concevoir une application web crud qui permet la gestion des annonces de vente de 
voitures en ligne.

                II-  PREREQUIS, DEPENDANCES ET DEPLOYEMENT
                  
 Pour executer ce projet, il est necessaire d'avoir les outils suivants deja installer :
   1 - PHP qui est le langage de programmation qui permet de communiquer avec la base de donnees
   2 - Le serveur xampp( sur les systeme de distribution linux) , wampp (sur les systemes windows) ou encore
    Mampp pour les distributions map.
   3 - Mysql qui est deja integre dans le serveur xampp ou wampp... donc plus besoin de l'installer.
   
  Notre projet utiise principalement en plus du langage php , le javascript , l'ajax , le html et le css.
  
  
                 III-   FONCTIONNEMENT DE L'APPLICATION
                    
   L'application fonctionne tel qu'il suit :
     - une simple connection a l'application vous donne acces aux annonces publies par les utilisateurs qui se   sont inscrit  dans l'application. 
    - Si l'on veut publier, supprimer ou modifier une annonce, il doit etre connecter et il ne peut que effectuer ces operations uniquement sur ces annonces qu'il a publies.
    -L'onglet recherche quant a elle permet de rechercher une voiture qui fait parti d'une annonce. La recherche se fait uniquement par filtrage de marque de voitures.  
     
                   IV-      REFERENCE
                              
    Pour concevoir cette application, on se base sur le fonction de l'application des annonces de ventes de voiture appelee : Auto Actu .
